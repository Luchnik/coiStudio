# coiStudio
COI Photo Studio front-end test project

To run project:
  - Clone the repository;
  - Install Bower and NPM globally;
  - Run
        ```
        $ bower install
        ```
        and
        ```
        $ npm install
        ```
        in project folder;
  -  **bower_components** folder must be inside **builds** folder
  -  Run project on **http-server** or similar;
